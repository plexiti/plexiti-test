Der Showcase zur Stellenausschreibung ist fertig*!

Es war ein schönes Stück Arbeit, aber es ist fürs erste geschafft - unsere Prozessanwendung zur Stellenausschreibung ist fertig!

Puuh, das war ein schönes Stück Arbeit, aber es ist fürs erste geschafft: unsere Prozessanwendung zur im Praxishandbuch BPMN 2.0 als Beispiel ausführlichst beschriebenen Stellenausschreibung ist fertig *! Unser Ziel ist nicht nur zum Selber spielen einzuladen, sondern auch eine auf camunda fox, Java EE 6 und Bootstrap basierende Prozessanwendung zu bauen, die einen Java Entwickler in die Lage versetzt binnen 60 Minuten mit der eigenen Anwendung zu starten: den Source Code kann man via GitHub herunterladen.

Wir sind auf dem Weg!

Unverhofft kommt oft! Wir erhielten ganz kurzfristig die schöne Gelegenheit unsere Prozessanwendung in Berlin vorzustellen.

Unverhofft kommt oft! Wir erhielten ganz kurzfristig die schöne Gelegenheit unsere Prozessanwendung zur im Praxishandbuch BPMN 2.0 beschriebenen Stellenausschreibung auf der BPMCon 2012 in Berlin vorzustellen. Und zwar im Rahmen des Pecha Kucha: 20 Folien zu je 20 Sekunden! Cool auch, weil wir nun keine Ausrede haben unsere Website einfach mal online zu stellen und “nach Berlin” dann weiterzubasteln. Release early, release often. Wir machen uns in mehrfacher Hinsicht “auf den Weg”!
